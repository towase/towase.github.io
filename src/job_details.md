## 2021/11/01 - 2022/06/30 : ロゴスウェア株式会社 GigaCast
企業向けウェビナー配信システムの開発および保守を担当した。

### 主な実績
- GigaCast 3大API群の1つを PHP から Golang に移行
  - 計画からパフォーマンステストまでの全てを主導した
  - フリーランスのエンジニアに2人ジョインしてもらい、コードレビューを主に行った
  - 既存の無駄な処理やデッドコードを削除して可読性を向上させた
  - テストを作成することでリファクタリングのハードルを下げた
- 一部のサーバーの CPU が一時的に80%を超える問題の原因を特定し、Socket.IO のブロードキャスト処理の負荷がデータ量にも依存することを証明した。その後負荷を削減した。
- セミナー映像の遅延の原因を突き止める
  - AWS SDK のデフォルトのタイムアウト時間が未設定であることを発見し、適切な値を検証して設定を反映した。これによりセミナーの堅牢性が増した。

### 得られた知識、スキル
- DDD
- コマンドクエリ分離原則
- OpenAPI
- Socket.IO の負荷仕様
- SDK の設定値をよく確認すること
